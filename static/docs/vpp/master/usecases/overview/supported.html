Architectures and Operating Systems¶

The following architectures and operating systems are supported in VPP:

Architectures¶

  • The FD.io VPP platform supports:

    • x86/64

    • ARM-AArch64

Operating Systems and Packaging¶

FD.io VPP supports package installation on the following recent LTS operating systems releases:

  • Operating Systems:

    • Debian

    • Ubuntu

    • CentOS

    • OpenSUSE
