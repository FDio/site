Downloading and Installing VPP

If you want to use VPP it can be convenient to install the binaries from existing packages. This guide describes how to pull, install and run the VPP packages.

This section provides directions on how to Install VPP binaries on Ubuntu, and Centos platforms.

FD.io VPP is installed using Package Cloud. For a complete set of instructions on how to install VPP with package cloud please refer to Package Cloud.
