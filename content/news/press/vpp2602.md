**VPP 26.02 Release on 25th of February 2026** -- The VPP 26.02 Release includes 32
new features such as Device driver updates, HTTP/3 and QPACK features, Host Stack applications,
Network Policies plugin, Shared Elog plugin, StateFul Data Plane Services plugin, and others.
More than 516 commits were merged since the previous release, including 123 fixes. Complete
list of features and bug fixes can be found in the Release Notes
[here](https://s3-docs.fd.io/vpp/26.02/aboutvpp/releasenotes/v26.02.html)!
